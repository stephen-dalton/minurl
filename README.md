minURL is a full-stack web application that allows you to shorten long URLs into concise, easy-to-share links. It consists of a React frontend built with Styled Components, and a Node.js backend powered by Express to handle URL shortening and redirection.

## Features

- Shorten long URLs with a single click.
- Redirect visitors to the original long URL when they access the shortened link.
